# 🧒 KID-GPT

A tiny language model I built from scratch that talks like a toddler.

Trained on conversations between two kids Max and Lily 
it generates dialogue like this:

```
Max: Snow!! Look look!
Lily: WHERE? I do not see!
Max: Outside window! White everything!
Lily: SNOW! We make snowman?
```

Built with Python and PyTorch. Trained for 25 mins on an RTX 2070.

This is my first ever AI project, one week into learning AI. Inspired by [Andrej Karpathy's GPT from scratch]
